# Pasta de Avatares

Coloque as fotos dos usuários nesta pasta com os seguintes nomes:

- `raphael.vascconcelos.jpg` (ou .png)
- `matheus.costa.jpg` (ou .png)
- `andrea.vasconcelos.jpg` (ou .png)
- `marcus.lopes.jpg` (ou .png)

**Formatos aceitos:** JPG, PNG, WEBP

**Tamanho recomendado:** 200x200px ou maior (quadrado)

As imagens serão exibidas automaticamente no sistema após serem adicionadas aqui.
